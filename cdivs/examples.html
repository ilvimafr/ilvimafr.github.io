1. Simple dividers

To create a simple divider between two blocks, place blocks with the data-cdivs attribute between the blocks between which you want to make a divider.
This will create a canvas between the blocks and draw the divider.

Blocks between which the divider is located must contain CSS background properties. Because the divider takes a background from these blocks to draw.

Note that the property background-attachment not supported and will be ignored.

2. Single block dividers

If there are elements inside the block that should be drawn on the divider but the divider draws only the background then use the data-cdivs-draw attribute or the draw option.

This allows you to draw the background of only one block. Instead of the background of another block, there will be transparency.

Use prev value to draw only the previous block, or next value to draw only the next block.

If there is only one block between the divider, then drawing one block will be automatic.

Note that the divider will shift towards the opposite block.

3. Animations

By default, animation is disabled. Use the data-cdivs-anim attribute to set animation for a specific divider or anim option to set animation for all dividers.
Use true value to enable animations or false value to disable.

To set the animation speed, use the separate data-cdivs-anim-duration attribute or the animDuration option.
This number will be multiplied by the animation speed. Initially, the animation duration is set to 2.0.

For mobile devices, animation is included as the separate mobAnim option.

4. z-index

If you need to change the CSS property of the z-index for the divider, you can use the data-cdivs-zindex attribute or the zIndex option. Initially is set to 2.

This will allow the divider to be on top of the blocks.

5. Overlays

If overlay is used in addition to the background then indicate which element is overlaid. Use :after, :before or css selector.

This option parses color and transparency from the element that you specify and will additionally be drawn on the divider.

In this example, an overlay is used which is made using the :after pseudo-class.

6. Inner backgrounds

If the background is an element that is in the block then you can use the data-cdivs-prev-bg and data-cdivs-next-bg attributes or the innerBg option.

This will allow you to specify the internal element that contains the background.

7. Divider inside block

If for some reason you cannot insert a divider between blocks, then place it inside one of the blocks. Then indicate in which block you inserted it. Use the data-cdivs-inside attribute or the inside option.

Use prev value to indicate that the divider is in the previous block, or next value indicate that the divider is in the next block.
