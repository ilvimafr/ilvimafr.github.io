Attributes

Additional attributes for customizing individual dividers

data-cdivs-size

Type: Number

Sets the size of divider in pixels.

data-cdivs-mob-size

Type: Number

Sets the size of divider in pixels for devices with small screen.

data-cdivs-offset

Type: Number

Shifts the divider to the specified percentage. Use a range of values between -100 to 100.

data-cdivs-anim

Type: Boolean
Values: true false

Adds light animation

data-cdivs-anim-duration

Type: Number

Animation duration will be multiplied by this number.

data-cdivs-draw

Type: String
Values: both prev next

Sets which blocks to draw. Use prev to draw only the previous block. Use next to draw only the next block. If use prev or next then the divider will shift towards the opposite block.

data-cdivs-zindex

Type: Number

Sets the value of css property z-index for divider.

data-cdivs-prev-overlay
data-cdivs-next-overlay

Type: String
Values: :after :before [css selector]

If overlay is used in addition to the background then indicate which element is overlaid. Use :after, :before or css selector. Css selector example: .overlay.

data-cdivs-prev-bg
data-cdivs-next-bg

Type: String
Values: :after :before [css selector]

If the background is inside the block then indicate which element is contains background. Use :after, :before or css selector. Css selector example: .inner-bg.

data-cdivs-inside

Type: String
Values: 'prev' 'next'

Use this attribute if you need to place a divider inside one of the blocks. Use prev if you placed a divider inside the previous block, next if you placed a divider inside the next block.

Options

List including all options.

size

Type: Number
Default: 100

Sets the size of all dividers in pixels.

mobSize

Type: Number
Default: 60

Sets the size of all dividers in pixels for devices with small screen.

mobAnim

Type: Boolean
Default: false

Adds light animation for all dividers on mobile devices.

mobScreenWidth

Type: Number
Default: 768

Sets the maximum width for devices with a small screen.

offset

Type: Number
Default: 0

Shifts all dividers to the specified percentage. Use a range of values between -100 to 100.

anim

Type: Boolean
Default: false

Adds light animation for all dividers.

animDuration

Type: Number
Default: 2.0

Animation duration will be multiplied by this number.

draw

Type: String
Values: 'both' 'prev' 'next'
Default: 'both'

Sets which blocks to draw. Use 'prev' to draw only the previous block. Use 'next' to draw only the next block. If use 'prev' or 'next' then the divider will shift towards the opposite block.

zIndex

Type: Number
Default: 2

Sets the value of css property z-index for all dividers.

overlay

Type: String
Values: ':after' ':before' '[css selector]'
Default: 'none'

If overlay is used in addition to the background then indicate which element is overlaid. Use ':after', ':before' or css selector. Css selector example: '.overlay'.

innerBg

Type: String
Values: ':after' ':before' '[css selector]'
Default: 'none'

If the background is inside the block then indicate which element is contains background. Use ':after', ':before' or css selector. Css selector example: '.inner-bg'.

inside

Type: String
Values: 'prev' 'next'
Default: false

Use this option if you need to place dividers inside one of the blocks. Use 'prev' if you placed dividers inside the previous block, 'next' if you placed dividers inside the next block.

blockZIndex

Type: String
Values: 'inherit' 'none'
Default: 'inherit'

Blocks inherit the z-index CSS property so that the divider inside can overlap the second block. Use the 'none' value to turn off inheritance.

Methods

JavaScript methods

CDivs.init(options)

Script initialization. Must be called once.

  • options - Object
    Optional parameter. All options are provided in the list.

CDivs.update()

Handles new dividers.

CDivs.refresh()

Updates dimensions and colors for existing dividers. Automatically called when resizing and loading a window.

[CanvasElement].cdRefresh()

Updates dimensions and colors for specific divider and blocks associated with it.

CDivs.add(name, offset, shapes)

Adds a new divider type.

  • name - String
    The name of the divider, which is then indicated in the attribute data-cdivs.
  • offset - Number
    Extra indent for divider.
  • shapes - Array
    Array of objects which contains:
    • opacity - Number

      Optional. Default - 1.0.

      Opacity of shape. From 0.0 to 1.0.

    • d - Array
      Contains shape coordinates. Coordinates are multiplied by dimensions. For example: ['M', 0,0, .1,1, .2,0, 'Q', .6,2, 1,1].
      • 'M', x,y - begins a new sub-path at the point specified by the given (x, y) coordinates.
      • 'L', x,y - adds a straight line to the current sub-path by connecting the sub-path's last point to the specified (x, y) coordinates.
      • 'A', x,y, radius, startAngle, endAngle - creates a circular arc centered at (x, y) with a radius of radius. The path starts at startAngle, ends at endAngle.
      • 'Q', x,y, x2,y2 - adds a quadratic Bézier curve to the current sub-path.
      • 'C', x,y, x2,y2, x3,y3 - adds a cubic Bézier curve to the current sub-path.
    • animEasing - String

      Optional. Default - 'quadraticEaseInOut'.

      Animation easing. Available values: 'linear', 'cubicEaseIn', 'cubicEaseOut', 'cubicEaseInOut', 'quadraticEaseIn', 'quadraticEaseOut' and 'quadraticEaseInOut'.

    • animDuration - Number

      Optional. Default - 2000.

      Duration for all animation keys in milliseconds.

    • animKeys - Array

      Optional.

      Contains information about animation keys. Should be array of objects which contains:

      • d - Array
        Contains shape coordinates. #d.
      • duration - Number

        Optional.

        Duration for specific animation key in milliseconds.

FAQ

What if a placed divider is not visible, or only half is visible?

Make sure the blocks between the divider contain css background properties.

Also, the reason may be a block that overlaps a divider. In this case, you need to set the z-index CSS property higher than the block that overlaps. Use the data-cdivs-zindex attribute or the zIndex option. Example.

What to do if an animation does not work?

By default, animation is disabled. Use the data-cdivs-anim attribute to set animation for a single divider or anim option to set animation for all dividers.

For mobile devices, animation is included as the separate mobAnim option. Example.

What if for some reason I can’t place a divider between blocks?

For this case, there is the data-cdivs-inside attribute or the inside option.

Place the divider inside one of the blocks and indicate in which block the divider is placed. Example.

What if a block has elements which the divider cannot draw?

If there are elements inside the block that should be drawn on the divider but the divider draws only the background then use the data-cdivs-draw attribute or the draw option.

This will allow you to draw only one block on the divider. Keep in mind that the divider will shift towards the opposite block so that the transparency of the divider does not overlap with the background of the block. Example.

What if an overlay is attached to the background?

For this case, there is the data-cdivs-prev-overlay and data-cdivs-next-overlay attributes or the overlay option.

This will allow you to specify an element using the css selector that contains an overlay. Example.

What if a background is in a separate element that is inside the block?

For this case, there is the data-cdivs-prev-bg and data-cdivs-next-bg attributes or the innerBg option.

This will allow you to specify an element using the css selector that contains a background. Example.
